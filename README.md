# Blog AAFMS – Mãos Que Acolhem

Blog oficial da **Associação de Amparo às Famílias de MS – Mãos Que Acolhem**.

## Estrutura

```
blog-aafms/
├── index.html              # Página inicial
├── sobre.html              # Sobre a associação
├── doe.html                # Página de doações
├── categoria.html          # Lista por categoria (a implementar)
├── assets/
│   ├── css/style.css       # Estilos principais
│   ├── js/main.js          # JavaScript
│   └── img/                # Imagens
└── posts/
    ├── bem-vindos.html
    ├── campanha-vacinacao.html
    ├── reforco-escolar.html
    ├── horta-comunitaria.html
    └── oficina-renda.html
```

## Como publicar no GitHub Pages

1. Faça o push deste repositório para o GitHub
2. Acesse **Settings → Pages**
3. Em **Source**, selecione `main` e a pasta `/ (root)`
4. Clique em **Save**
5. O blog ficará disponível em: `https://adrianouperttech.github.io/blog-aafms/`

## Como adicionar um novo post

1. Crie um novo arquivo HTML na pasta `posts/` (copie um existente como base)
2. Edite o conteúdo do post
3. Adicione o card do post no `index.html` dentro da seção `posts-mini-grid`
4. Faça o commit e push

## Tecnologias

- HTML5 semântico
- CSS3 puro (sem frameworks)
- JavaScript vanilla
- Google Fonts: Lora + DM Sans
- Zero dependências externas
